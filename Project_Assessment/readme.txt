In order to operate this program. You will need to use microsoft visual studio 2019 in order to run and operate.
In terms of compiling, just the use the visual studio 2019 debugging tool in order to compile.

Also, there are secret commands in order to get to the extra parts of the program.
These secret commands are (superuser) and (secretstash).
Superuser is where you can add in a new item to the shop and it will be placed into the secretstash inventory to buy.
Secretstash requires a password in order to operate. This password is (ForHonor) all in one word.
This allows you to get access to another part of the shop that you would not have been able to access before with all the newly added items.
